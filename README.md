Download files

Open CMD or bash and join this folder using this command
```
cd <script-folder-path>
```
Setup virtualenv
```
python -m venv venv
```
Activate it:

On windows
```
venv\Script\activate
```
MacOS or Linux:
```
source venv/bin/activate
```
Download requirements:
```
pip install -r requirements.txt
```
Run the script
```
python checker.py
```
